This demo release shows how to announce infrastructure and developer-experience improvements in the changelog.

### Features

- **One-command workspace setup** - New projects can start their app and backend services from the same command.
- **Clearer environment checks** - Missing configuration is surfaced earlier with focused messages.
- **Reusable setup scripts** - Local agents and developers can share the same project bootstrap path.

### Improvements

- **Cleaner onboarding path** - Setup steps are grouped around the order developers actually follow.
- **Reduced template noise** - Optional integrations stay out of the way until they are configured.
