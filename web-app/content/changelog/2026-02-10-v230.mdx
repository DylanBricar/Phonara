This demo entry highlights a product-facing release with dashboard updates, reporting improvements, and small workflow fixes.

### Features

- **Activity overview cards** - Teams can scan key account and usage signals from the main dashboard.
- **Feedback trend chart** - Admins get a visual breakdown of new feedback over time.
- **Member activity snapshot** - Organization owners can quickly understand how their team is using the product.

### Bug Fixes

- **Stable loading states** - Dashboard sections now keep their layout while data is loading.
- **Better empty states** - Empty dashboards explain the next useful action without adding extra setup steps.
